# Sergio Neto - Currículo Online

Este é o repositório do meu currículo online pessoal, hospedado via GitHub Pages.

## Sobre o Projeto
Este projeto consiste em uma página web estática construída com o objetivo de apresentar de forma profissional e interativa:
- Minha trajetória profissional.
- Educação e certificações.
- Habilidades técnicas (Programação e Infraestrutura).
- Depoimentos.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (jQuery)
- Bootstrap
